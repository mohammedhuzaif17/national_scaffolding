## HOW TO USE THIS CODE

1. **Add CSS to your <style> tag in the template**
   - All the CSS above is included in national_scaffoldings.html
   - It contains all responsive styles for different screen sizes

2. **Add HTML structure**
   - The hero-section div is inserted before the main page-container
   - It appears at the very top of the page
   - Content is centered and animated on load

3. **Image Path**
   - Default path: static/images/scaffolding-hero.jpg
   - Flask auto-generates the correct URL using url_for()
   - Just save your image with the correct name in that folder

4. **Customization**
   - Change hero-section min-height for different heights
   - Adjust rgba(0, 29, 61, 0.65) values for overlay darkness
   - Modify font sizes in hero-content h1, h3, etc.
   - Change button colors by modifying #ffd966 (gold) values
