## ğŸ¬ Design Philosophy

This CSS is inspired by Netflix's ultra-clean, minimal design philosophy:

### Netflix Design Principles Applied:

1. **Simplicity Over Complexity**
   - Removed all gradients and complex effects
   - Clean, solid colors only
   - Minimal shadows (only for depth)

2. **Bold Typography**
   - Large, bold headings
   - Negative letter-spacing for modern look
   - Clear hierarchy

3. **Minimal Animations**
   - Simple 0.2s transitions
   - No fancy effects
   - Smooth, subtle movements

4. **Clean Spacing**
   - Consistent padding/margins
   - Breathing room between elements
   - Clear visual grouping

5. **Simple Button Styles**
   - Solid fills (no gradients)
   - Clear hover states
   - 4px border radius (subtle corners)

## ğŸ“ Design Specifications

### Typography:
```
Font Family: -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto...
Headings: 700 weight, -0.5px letter spacing
Body: 500 weight, normal spacing
```

### Spacing:
```
Small: 0.5rem (8px)
Medium: 1rem (16px)
Large: 2rem (32px)
Gaps: 1.5rem (24px)
```

### Shadows:
```
Default: 0 2px 8px rgba(0, 0, 0, 0.08)
Large: 0 4px 20px rgba(0, 0, 0, 0.12)
```

### Border Radius:
```
Everything: 4px or 8px
No rounded pills
```

### Transitions:
```
All: 0.2s (fast, like Netflix)
No complex easing
```
