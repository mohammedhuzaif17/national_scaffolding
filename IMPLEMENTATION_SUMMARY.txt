================================================================================
                            CUSTOMIZATION OPTIONS
================================================================================

Want to adjust the appearance? Here are the options:

1. CHANGE HERO HEIGHT:
   File: templates/national_scaffoldings.html
   Find: min-height: 450px;
   Change to: min-height: 500px; (or any size you want)

2. CHANGE OVERLAY DARKNESS:
   Find: rgba(0, 29, 61, 0.65)
   Change 0.65 to:
   - 0.3 = lighter overlay (more image visible)
   - 0.8 = darker overlay (more text readable)

3. DISABLE PARALLAX EFFECT:
   Find: background-attachment: fixed;
   Change to: background-attachment: scroll;

4. CHANGE BUTTON TEXT/COLOR:
   Find: "Go to Fabrications" button
   Modify text and background color (#ffd966 = gold)

5. USE DIFFERENT IMAGE:
   Replace "scaffolding-hero.jpg" filename in the URL
